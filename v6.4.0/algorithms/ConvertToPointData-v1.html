Summary¶

Converts a workspace containing histogram data into one containing point data.

See Also¶

ConvertToHistogram

Properties¶

Name Direction Type Default Description
InputWorkspace Input MatrixWorkspace Mandatory Name of the input workspace.
OutputWorkspace Output MatrixWorkspace Mandatory Name of the output workspace, can be the same as the input.

Description¶

The InputWorkspace must contain histogram data. Once executed the OutputWorkspace will contain point data, where the X values are simply the centre points of the input bins.

Note that when this is applied to a RebinnedOutput workspace created by SofQWNormalisedPolygon v1 the bin fractions information is discarded (set to 1) even if the workspace is subsequently converted back to a histogram. This means that error propagation after conversion may not be correct.
