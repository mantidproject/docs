GausOsc¶

Description¶

Oscillating Gaussian decay for use by Muon scientists defined by

\[\mbox{A}e^{-\sigma^2 x^2}\cos(2\pi f x + \phi)\]

Properties (fitting parameters)¶

Name Default Description
A 0.2 Amplitude at time 0
Sigma 0.2 Decay rate
Frequency 0.1 Frequency of oscillation
Phi 0.0 Frequency of oscillation
