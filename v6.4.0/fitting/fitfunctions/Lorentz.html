Lorentz¶

Description¶

The Ornstein-Zernicke model is defined as:

\[I(q) = I_0 / (1 + q^2L^2) + background\]

where: - \(I_0\) is a scale factor - \(L\) is the screening length - \(background\) is the background level

Properties (fitting parameters)¶

Name Default Description
Scale 1.0 Scale
Length 50.0 Length
Background 0.0 Background
