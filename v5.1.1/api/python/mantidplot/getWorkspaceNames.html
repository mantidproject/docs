getWorkspaceNames¶

pymantidplot.proxies.getWorkspaceNames(source)¶

Takes a “source”, which could be a WorkspaceGroup, or a list of workspaces, or a list of names, and converts it to a list of workspace names.

Args:
source :: input list or workspace group
Returns:
list of workspace names
