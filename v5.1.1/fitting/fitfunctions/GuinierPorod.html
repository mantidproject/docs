GuinierPorod¶

Description¶

Calculates the scattering for a generalized Guinier/power law object. The Guinier-Porod model is defined in Hammouda, J. Appl. Cryst. (2010) 43, 716-719.

Properties (fitting parameters)¶

Name Default Description
Scale 1.0 Scale
Dimension 1.0 Dimension
Rg 50.0 Radius of gyration
M 3.0 M
