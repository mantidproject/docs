Abragam¶

Description¶

Abragam fitting function for use by Muon scientists defined by

\[\mbox{A}\cos(2 \pi \omega x + \phi) \times \exp\left(-\sigma^2 \tau^2 x^2 \left(e^{\frac{x}{\tau}} -1 + \frac{x}{\tau}\right)\right)\]

Properties (fitting parameters)¶

Name Default Description
A 0.2 Amplitude
Omega 0.5 Angular Frequency of oscillation
Phi 0.0 Phase of oscillation at 0 (in Radians)
Sigma 1.0 Decay rate?
Tau 1.0 ?
