ThermalNeutronBk2BkExpSigma¶

Description¶

ThermalNeutronBk2BkExpSigma is a function to calculate \(\sigma(d_h)\) of ThermalNeutronBk2BkExpConvPVoigt. It is defined as

\[\sigma(d_h) = \left(\sigma_0^2 + \sigma_1^2 + d_h^2 + \sigma_2^2 + d_h^4\right)^{0.5}\]
