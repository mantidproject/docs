EndErfc¶

Description¶

This is a fitting function for the complementary error function.

The function is A*erfc((B-x)/C) + D for positive A and the same minus 2A for negative A.

Properties (fitting parameters)¶

Name Default Description
A 2000.0 Half value at minus infinity minus half value at plus infinity
B 50.0 Mid x value
C 6.0 Width parameter
D 0.0 Minimum value - must not be negative
