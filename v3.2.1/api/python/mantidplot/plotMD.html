plotMD¶

mantidplot.plotMD(source, plot_axis=-2, normalization=1, error_bars=False, window=None, clearWindow=False)¶

Open a 1D plot of a MDWorkspace.

Args:
source: Workspace(s) to plot plot_axis: Index of the plot axis (defaults to auto-select) normalization: Type of normalization required (defaults to volume) error_bars: Flag for error bar plotting. window: window used for plotting. If None a new one will be created clearWindow: if is True, the window specified will be cleared before adding new curve
Returns:
A handle to the matrix containing the image data.
