UnGroupWorkspace v1¶

UnGroupWorkspace dialog.

Table of Contents

  • Summary
  • Properties
  • Description

Summary¶

Takes a group workspace as input and ungroups the workspace.

Properties¶

Name Direction Type Default Description
InputWorkspace Input string Mandatory Name of the input workspace to ungroup

Description¶

Takes a WorkspaceGroup as input and ungroups it into several workspaces. You can perform this from the MantidPlot GUI by selecting the WorkspaceGroup and clicking “Ungroup”.
