Nexus File¶

A Nexus file is a type of data file used by ISIS traget station 2 and by MantidPlot. The format appears to be like an XML file plus some unprintable characters.

ManditPlot is capable of saving certain types of workspace as a Nexus file. It can also save a project as a mantid file plus Nexus files.
