Summary¶

Calculate the Q resolution for TOF SANS data for each pixel.

Properties¶

Name Direction Type Default Description
Workspace InOut MatrixWorkspace Mandatory Name the workspace to calculate the resolution for, for each pixel and wavelenght
DeltaR Input number 0 Pixel size (mm).
SampleApertureRadius Input number 0 Sample aperture radius (mm).
SourceApertureRadius Input number 0 Source aperture radius (mm).
SigmaModerator Input MatrixWorkspace Mandatory Sigma moderator spread in units of microsec as a function of wavelenght.
