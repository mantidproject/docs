EnginXCalibrateFull v1¶

Summary¶

Calibrates every pixel position by performing single peak fitting.

Properties¶

Name Direction Type Default Description
Filename Input string Mandatory Calibration run to use
DetectorPositions Output TableWorkspace Mandatory A table with the detector IDs and calibrated detector positions in V3P format.
ExpectedPeaks Input dbl list   A list of dSpacing values where peaks are expected.
Bank Input number 1 Which bank to calibrate

Description¶

Warning

This algorithm is being developed for a specific instrument. It might get changed or even removed without a notification, should instrument scientists decide to do so.

Allows to correct for tiny variations in pixel parameters. It does this by fitting the peak for each of the the selected bank’s indices, (using EnginXFitPeaks v1 ) and using the resulting difc value to calibrate the detector position. The calibrated detector position is produced as shown below:

The cartesian2d vector is then returned for the given spherical co ordinates (L2, 2theta and phi).

The result of the calibration is accepted by both EnginXCalibrate v1 and EnginXFocus v1 to correct the detector positions before focussing.

Expects the long calibration run, which provides a decent pattern for every pixel.

Usage¶

Note

To run these usage examples please first download the usage data, and add these to your path. In MantidPlot this is done using Manage User Directories.

Example - Calculate Difc and Zero:

table = EnginXCalibrateFull(Filename="ENGINX00213855focussed.nxs",
                                  ExpectedPeaks=[1.097, 2.1], Bank=1)

pos =  table.column(1)[0]
print "Det ID:", table.column(0)[0]
print "Calibrated position: (%.3f,%.3f,%.3f)" % (pos.getX(),pos.getY(),pos.getZ())

Output:

Det ID: 100001
Calibrated position: (1.506,0.000,0.002)
