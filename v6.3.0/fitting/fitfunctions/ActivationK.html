ActivationK¶

Description¶

The ActivationK fitting function can be written as:

\[y = A_R e^{-\frac{b}{x}}\]

where: - A_R - Attempt Rate - b - Barrier energy - x - is in Kelvin

Alternatively use the ActivationmeV <func-ActivationmeV> fitting function when fitting to meV.

Examples¶

An example of when this might be used is for examining the ion dynamics in flouride-containing polyatomic anion cathodes using muon spectroscopy [1].

Properties (fitting parameters)¶

Name Default Description
AttemptRate 1000.0 coefficient for attempt rate
Barrier 1000.0 coefficient for barrier energy

References¶

[1] Johnston, B. I., Baker, P. J. & Cussen, S. A. (2021). Ion dynamics in flouride-containing polyatomic anion cathodes by muon spectroscopy. Journal of Physics:Materials, Vol. 4 No. 4, https://iopscience.iop.org/article/10.1088/2515-7639/ac22ba.
