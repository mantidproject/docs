ComptonScatteringCountRate¶

Description¶

The ComptonScatteringCountRate function fits Voigt functions convolved with VesuvioResolutions to input data subject to a set of supplied constraints. The constraints are supplied through function attributes:

Attributes (non-fitting parameters)¶

Name Type Default Description
BackgroundOrderAttr      
IntensityConstraints      
NumDeriv      

These constraints include positivity restrictions on the Compton profile \(J(y)\) or equality constraints.
