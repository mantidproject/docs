Properties¶

Name Direction Type Default Description

Verbose Input boolean False Output more verbose message to log
Plot Input boolean False Plot result workspace
Save Input boolean False Save result workspace to nexus file in the default save directory

Description¶

Performs jump fitting on a workspace created by either ConvFit (Indirct Data Analysis) or Quasi (Indirect Bayes).

For more details, see the Indirect Bayes docs.
