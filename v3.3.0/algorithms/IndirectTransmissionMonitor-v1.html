IndirectTransmissionMonitor v1¶

IndirectTransmissionMonitor dialog.

Table of Contents

  • Summary
  • Properties
  • Description
  • Usage

Summary¶

Calculates the sample transmission using the raw data files of the sample and its background or container.

Properties¶

Name Direction Type Default Description
SampleWorkspace Input Workspace Mandatory Sample workspace
CanWorkspace Input Workspace Mandatory Background/can workspace
OutputWorkspace Output Workspace Mandatory Output workspace group
Verbose Input boolean False Output more verbose message to log
Plot Input boolean False Plot result workspace
Save Input boolean False Save result workspace to nexus file in the default save directory

Description¶

Calculates the sample transmission using the raw data files of the sample and its background or container.

For more details, see Indirect:Transmission.

Output workspace will default to [instument][run number]_[analyser][reflection]_Transmission if not set.

Usage¶

Example - Create mapping file for IRIS

sample_file = 'IRS26176.RAW'
can_file = 'IRS26173.RAW'

sample_ws = Load(sample_file)
can_ws = Load(can_file)

transmission_ws = IndirectTransmissionMonitor(SampleWorkspace=sample_ws, CanWorkspace=can_ws)

print transmission_ws.getNames()

Output:

['sample_ws_Sam','sample_ws_Can','sample_ws_Trans']
