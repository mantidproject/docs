LoadFITS dialog.

Table of Contents

  • Summary

Summary¶

Load data from FITS files.

Properties¶

Name Direction Type Default Description
Filename Input list of str lists Mandatory The input filename of the stored data
OutputWorkspace Output Workspace Mandatory  
ImageKey Input number -1 Image type to set these files as. 0=data image, 1=flat field, 2=open field, -1=use the value from FITS header.
BitDepthName Input string BITPIX Name for the pixel bit depth header key.
RotationName Input string ROTATION Name for the rotation header key.
AxisNames Input string NAXIS1,NAXIS2 Names for the axis header keys, comma separated string of all axis.
ImageKeyName Input string IMAGEKEY Names for the image type, key.
HeaderMapFile Input string   A file mapping header keys to the ones used by ISIS [line separated values in the format KEY=VALUE, e.g. BitDepthName=BITPIX

Description¶

Load data from FITS files
