Levenberg-Marquardt MD MinimizerÂ¶

This minimizer is the same as the Levenberg-Marquardt minimizer as explained in - Wikipedia , but is intended for MD workspaces or for a large number of data points. It divides its work into chunks to achieve a greater efficiency for a large number of data points than can be obtained from the default Levenberg-Marquardt minimizer.

It is listed in a comparison of fitting minimizers.

It makes use of the GSL (GNU Scientific Library) library, specifically the GSL routines for least-squares fitting.
