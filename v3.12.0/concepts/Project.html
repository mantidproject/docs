Project¶

In MantidPlot, you can save your work into project. Then when you run MantidPlot at a later time you can recover this work by loading the project.

What a Project Consists of¶

A project consists of a .mantid file and a collection of Nexus files it refers to. For this reason, a project is put into its own folder when saved.
