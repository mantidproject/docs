importMatrixWorkspace¶

mantidplot.importMatrixWorkspace(name, firstIndex=None, lastIndex=None, showDialog=False, visible=False)¶

Create a MantidMatrix object from the named workspace.

Args:
name: The name of the workspace. firstIndex: The workspace index to start at (default: the first one). lastIndex: The workspace index to stop at (default: the last one). showDialog: Whether to bring up a dialog to allow options to be entered manually (default: no). visible: Whether to initially show the created matrix (default: no).
Returns:
A handle to the created matrix.
