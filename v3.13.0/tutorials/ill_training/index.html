ILL Training¶

Welcome to the ILL Mantid training course. This course is approximately 2 hours long and gives a basic introduction to using Mantid for data reduction at the ILL.

Outline¶

The training course is split into four sections:
  1. Introduction
  2. MantidPlot Basics
  3. MantidPlot, Algorithms and Workspaces
  4. Using Mantid with Python

This is only a short introduction, more complete training courses are available on the Mantid website.
