InelasticDiffSphere¶

Description¶

This fitting function models the inelastic part of the dynamics structure factor of a particle undergoing continuous diffusion but confined to a spherical volume, DiffSphere.

Because of the spherical symmetry of the problem, the structure factor is expressed in terms of the spherical Bessel functions.

The value of the momentum transfer can be obtained either through attribute , or can be calculated from the input workspace using attribute . The value calculated using the workspace is used whenever attibute is set empty.
